# MapTap Tracker — Suspects

Live leaderboard for the Suspects group chat MapTap games.

**View the dashboard:** https://jgleason923.github.io/maptap-tracker

## About

This tracker logs daily MapTap scores for James, Joey, Greg, Brendan, Connor, and Mark. The dashboard includes:
- All-time records and leaderboard
- Daily winners and results
- Score trends over time
- Complete game history

Data syncs automatically from the Excel tracker.
